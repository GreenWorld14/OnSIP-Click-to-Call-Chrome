http://code.google.com/chrome/extensions/autoupdate.html
using this text file
Updating the extension on Chrome gallery:
* create a ZIP archive with the source files
* make sure you increment the extension's version number in the manifest.json file
* make sure there is no update_url in the manifest.json file
* upload it on the developer dashboard page
